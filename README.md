
# **Jogos com Python - Aula**

Este repositório contém **dois jogos** desenvolvidos em **Python** utilizando a biblioteca **Pygame**.
Os jogos são simples e divertidos, focando em mecânicas clássicas para a prática de programação e desenvolvimento de jogos.

 **Jogos Inclusos:**

1. **Acerte o Alvo** (`acerte-alvo.py`)


3. **Pong** (`pong.py`)
   




---

## **Como Executar os Jogos**

### **Pré-requisitos**

Para executar os jogos, você precisará do **Python 3.x** instalado e da biblioteca **Pygame**.

#### 1. Instale o Python

Baixe e instale o Python 3.x através do site oficial: [https://www.python.org/downloads/](https://www.python.org/downloads/)

#### 2. Instale o Pygame

Abra o terminal (ou prompt de comando) e execute o seguinte comando para instalar o Pygame:

```bash
pip install pygame
```

---

### **Baixando o Projeto**

Clone o repositório para o seu computador:

```bash
git clone https://github.com/NathyBruBru/Python-Jogos-Aula.git
```

Entre no diretório do projeto:

```bash
cd Python-Jogos-Aula
```

---

### **Jogando os Jogos**

#### 1. **Acerte o Alvo**

Inicie o jogo com o seguinte comando:

```bash
python acerte-alvo.py
```

**Descrição:**
O objetivo do jogo é clicar no alvo que aparece na tela para ganhar pontos.
